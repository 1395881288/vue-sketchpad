# sketchpad

>使用vue.js和fabric.js构建的web画板，中间使用vuex来做状态的控制，目前支持画箭头、虚线、矩形、原型、文字等功能，删除功能和回退功能待开发中

效果图如下：
![](src/assets/img.png)

- [x] 直线
- [x] 箭头
- [x] 虚线
- [x] 输入文字
- [x] 矩形
- [x] 标准圆形
- [x] 椭圆形
- [x] 等腰三角形
- [x] 删除、移动、复位
- [x]  历史记录

2019-11-30更新 修复了回撤前进的bug、支持Retina屏 重新整理了下代码组织
